# Security Assessment

In 2026, this project was revisited as an Application Security (AppSec) case study.

The assessment included:

- Architecture Review
- Android Lifecycle Analysis
- Attack Surface Analysis
- STRIDE Threat Modeling
- Manual Validation of Findings
- Risk Assessment
